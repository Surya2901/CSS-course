17. #CSS Positioning(3Diff options: Absolute, Relative, Fixed positioning)
*Absolute Positioning: An element is positioned at the specified coordinates relative to your screen top left corner.
Two values top and left can be used along with the position property to move an HTML element.
*Relative Positioning: changes the position of the HTML element relative to where it normally appears.
*Fixed Positioning: Allows you to fix the position of an element on the page, regardless of scrolling.

<!DOCTYPE html>
